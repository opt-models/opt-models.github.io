## opt-models.org

This repository houses the [opt-models.org](https://opt-models.org/) website which uses the [Hugoplate](https://themes.gohugo.io/themes/hugoplate/) theme and the [Hugo](https://gohugo.io/) static site generator, and is hosted on [GitHub Pages](https://pages.github.com/).

# Note!
The actual models and app-building code featured on the opt-models.org site are housed in a separate repo at https://github.com/opt-models/opt-models.
